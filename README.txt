¡¡¡¡SHOPPING LIST !!!!

app donde se puede listar productos con sus precios y donde muestra el total de la compra en vivo.
el producto puede tener 3 instancias 
    pausa: se agrego pero todavia no se confirma su compra
    check:el producto se confirmo para la compra y se sumara al total,
    eliminado:el producto ya no existe en la lista
cuenta con 4 opciones
    add product: agrega el producto en instancias de pausa
    new list: limpia la lista
    Retrieve list: recupera todos los productos confirmados en instancia de pausa
    dark mode/ligth Mode: modo oscuro/modo claro
